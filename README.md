
## Setup & Run Instructions

1. Install project dependencies:

   ```bash
   npm install
````

2. Run database migrations:

   ```bash
   npx sequelize-cli db:migrate
   ```

3. Seed the database:

   ```bash
   npx sequelize-cli db:seed:all
   ```

4. Start the project:

   ```bash
   npm start
   ```
